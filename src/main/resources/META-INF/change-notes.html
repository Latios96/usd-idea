Version 0.3:
  • Added initial support for structure view!
    This allows you to view the outline/structure of USD files and navigate quickly to objects using the Structure tool view.
    The current list of supported objects in Structure view are:
    • Prims
    • Attribute properties
    • (Some) relationship properties
    The objects' name & type (if available) will be displayed in the Structure.
  • New build for newer 2020.* version of IDEA IDEs
